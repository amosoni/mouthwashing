# 缄默祸运 - 游戏官方网站

这是一个基于《Mouthwashing》游戏的心理恐怖主题网站。

## 项目特点

- 🎮 心理恐怖游戏主题
- 🎨 现代响应式设计
- ✨ 流畅的动画效果
- 📱 移动端适配
- 🌟 优雅的视觉效果

## 快速开始

### 方法一：使用 npm（推荐）

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 打开浏览器访问：`http://localhost:3000`

### 方法二：直接打开

直接双击 `index.html` 文件在浏览器中打开。

### 方法三：使用 Python 简单服务器

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后访问：`http://localhost:8000`
